Rattrapage de Lucie 

## ✨ Les Touches

| Action              | Clavier                             | Contrôle        |
| :------------------ | :---------------------------------- | :-------------- |
| **Direction**       | `W` `A` `S` `D` ou `Arrow Keys`     | `D-Pad`         |
| **Shoot**           | `Space`                             | `Right Trigger` |
| **Menu**            | `Up` / `Down` and `Enter` / `Space` | `D-Pad` et `A` |
| **Pause / Retour**  | `ESC`                               | `Start` / `B`   |
